서비스 소개

Perplexity는 AI와 검색 엔진을 결합한 서비스입니다. 질문을 입력하면 웹을 실시간으로 검색해 관련 출처를 인용하면서 요약 답변을 제공합니다. 기존 검색 엔진처럼 링크 목록을 나열하는 대신, AI가 정보를 종합해 직접 답변하는 방식으로 조사·리서치 작업에 특히 유용합니다.

주요 기능

  • 🔍 실시간 웹 검색
  • 📌 출처·인용 표시
  • 📄 문서·PDF 검색
  • 🤖 GPT-4·Claude 연동
  • 📊 학술 논문 검색
  • 💬 후속 질문 지원
  • 📱 모바일 앱 제공
  • 🆓 무료 사용 가능

함께 보면 좋은 사이트

🧠ChatGPT ✨Gemini 🤖Claude 🎨Midjourney

💡 Tips

  • 출처가 표시되므로 정보의 신뢰성을 직접 확인할 수 있습니다. 중요한 내용은 원문 링크를 클릭해 검증하세요.
  • Perplexity Pro는 GPT-4나 Claude 같은 고급 모델을 선택해 답변 품질을 높일 수 있습니다.
  • 후속 질문(follow-up)을 연속으로 해서 특정 주제를 점점 더 깊게 파고들 수 있습니다.

자주 묻는 질문

무료로 사용할 수 있나요?

기본 기능은 무료입니다. 고급 AI 모델 선택과 파일 업로드 등 추가 기능은 Perplexity Pro($20/월)로 이용 가능합니다.

일반 구글 검색과 다른 점은?

구글은 링크 목록을 보여주지만, Perplexity는 AI가 여러 출처를 종합해 직접 답변을 생성합니다. 복잡한 질문에 더 빠른 답변을 얻을 수 있습니다.

정보가 항상 정확한가요?

실시간 웹 검색 기반이라 최신 정보를 제공하지만, AI가 생성한 요약이므로 중요한 정보는 반드시 원문 출처를 확인하세요.
