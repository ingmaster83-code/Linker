서비스 소개

OneDrive는 Microsoft가 제공하는 클라우드 스토리지 서비스입니다. Windows에 기본 내장되어 있어 별도 설치 없이 바로 사용할 수 있으며, Microsoft 365(Word·Excel·PowerPoint)와 완벽하게 통합됩니다. 5GB 무료 제공이며, Microsoft 365 구독 시 1TB까지 확장됩니다. PC 폴더를 자동으로 백업하고 어디서나 파일을 접근할 수 있습니다.

주요 기능

  • 🔷 5GB 무료 저장 공간
  • 📄 Word·Excel·PowerPoint 연동
  • 🖥️ Windows 기본 내장
  • 🔄 PC 폴더 자동 백업
  • 👥 실시간 공동 편집
  • 🔗 링크 공유
  • 📱 모바일 앱 지원
  • 🔒 개인 볼트(보안 폴더)

💡 Tips

  • Windows 설정 → OneDrive → "중요 PC 폴더 백업"을 활성화하면 바탕화면·문서·사진 폴더가 자동으로 클라우드에 백업됩니다.
  • 개인 볼트(Personal Vault) 기능을 사용하면 중요한 파일을 추가 인증(PIN·지문) 없이는 접근할 수 없게 보호할 수 있습니다.
  • Microsoft 365 Personal/Family 구독 시 1TB 저장 공간이 제공되어 Google One보다 경제적일 수 있습니다. Office 앱이 필요하다면 함께 고려해보세요.

자주 묻는 질문

Google Drive와 어떤 차이가 있나요?

OneDrive는 Windows에 기본 통합되어 Word·Excel·PowerPoint와 함께 사용할 때 강점이 있고, Google Drive는 Google Docs·Sheets·Slides와 연동이 뛰어납니다. 이미 사용 중인 생태계에 맞게 선택하면 됩니다.

무료 5GB는 어디에 쓸 수 있나요?

OneDrive에 저장하는 모든 파일에 사용됩니다. Microsoft 365 구독이 없어도 Office 온라인(웹 버전)으로 Word·Excel·PowerPoint 파일을 편집할 수 있습니다.

파일을 실수로 삭제했어요. 복구할 수 있나요?

삭제한 파일은 휴지통에 30일간 보관됩니다. 또한 OneDrive는 파일 버전 기록을 저장하므로, 이전 버전으로도 복원이 가능합니다.
