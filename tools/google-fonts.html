서비스 소개

Google Fonts는 Google이 제공하는 무료 웹폰트 서비스입니다. 1500개 이상의 폰트 패밀리를 CSS @import 또는 link 태그 한 줄로 웹사이트에 적용할 수 있습니다. 모든 폰트는 오픈 라이선스(OFL 등)로 개인·상업 모두 무료로 사용할 수 있으며, 한국어 폰트도 다수 포함되어 있습니다.

주요 기능

  • 🔤 1500+ 폰트 패밀리
  • 🆓 완전 무료·상업 이용
  • 💻 CSS 한 줄 적용
  • 👁️ 실시간 미리보기
  • 🌍 한국어 포함 다국어
  • 📥 폰트 파일 다운로드
  • 🎨 폰트 페어링 추천
  • 📊 인기순·최신순 정렬

💡 Tips

  • 폰트 선택 시 'Latin' 외 'Korean' 서브셋을 선택해야 한국어가 정상적으로 표시됩니다.
  • 웹 성능을 위해 폰트 굵기(weight)는 실제 사용할 것만 선택하세요. 불필요한 weight를 포함하면 로딩 시간이 늘어납니다.
  • 'Pairings' 탭에서 어울리는 폰트 조합을 추천받을 수 있어 폰트 선택에 도움이 됩니다.

자주 묻는 질문

완전 무료인가요?

네, 개인·상업 프로젝트 모두 완전 무료입니다. 별도 라이선스 구매 없이 웹·인쇄·영상 등 모든 매체에 사용할 수 있습니다.

폰트 파일을 다운로드할 수 있나요?

네, 각 폰트 페이지에서 TTF/OTF 파일을 무료로 다운로드해 로컬 설치나 오프라인 프로젝트에도 사용할 수 있습니다.

한국어 폰트도 있나요?

나눔고딕·나눔명조·Noto Sans KR·Noto Serif KR·검은고딕 등 다수의 한국어 폰트가 포함되어 있습니다.
