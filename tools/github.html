서비스 소개

GitHub은 Git 기반의 코드 호스팅 플랫폼으로, 전 세계 1억 명 이상의 개발자가 사용하는 협업 허브입니다. 코드 버전 관리(commit/branch/merge), 이슈·PR 기반 협업, CI/CD 자동화(GitHub Actions), 오픈소스 프로젝트 참여까지 소프트웨어 개발의 전 과정을 지원합니다.

주요 기능

  • 📦 Git 저장소 호스팅
  • 🔀 브랜치·PR 관리
  • 🐛 이슈 트래킹
  • ⚙️ GitHub Actions (CI/CD)
  • 🤖 GitHub Copilot
  • 📊 코드 리뷰
  • 🌐 오픈소스 허브
  • 📄 GitHub Pages

💡 Tips

  • README.md 파일을 잘 작성하면 오픈소스 프로젝트의 첫인상이 좋아집니다. 설치 방법·사용법·기여 방법을 명확히 기재하세요.
  • GitHub Actions를 활용하면 push 시 자동으로 테스트·빌드·배포가 실행되는 CI/CD 파이프라인을 무료로 설정할 수 있습니다.
  • GitHub Pages로 정적 웹사이트를 무료로 호스팅할 수 있습니다. HTML/CSS/JS 프로젝트나 문서 사이트에 유용합니다.

자주 묻는 질문

무료로 사용할 수 있나요?

개인 무료 플랜에서 무제한 공개·비공개 저장소를 제공합니다. 팀 협업 기능 강화나 고급 보안 기능은 유료 플랜(Team $4/월/사용자~)이 필요합니다.

Git과 GitHub의 차이는?

Git은 로컬 컴퓨터에서 동작하는 버전 관리 시스템이고, GitHub은 Git 저장소를 온라인에서 호스팅하고 협업 기능을 추가한 서비스입니다.

비공개 저장소도 만들 수 있나요?

네, 무료 플랜에서도 비공개(Private) 저장소를 무제한으로 만들 수 있습니다.
