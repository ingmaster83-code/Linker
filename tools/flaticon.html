서비스 소개

Flaticon은 세계 최대 벡터 아이콘 라이브러리입니다. 1600만 개 이상의 아이콘을 SVG·PNG·PSD·EPS·웹폰트 형식으로 다운로드할 수 있습니다. 스타일별·카테고리별 검색이 가능하며, 색상·크기를 온라인 에디터에서 즉시 수정할 수 있습니다.

주요 기능

  • 🎨 1600만+ 아이콘
  • 📥 SVG·PNG·PSD 다운로드
  • ✏️ 온라인 색상 편집
  • 🔠 아이콘 폰트
  • 🗂️ 팩 단위 다운로드
  • 🔍 스타일별 검색
  • 🆓 무료 사용
  • 🔗 API 제공

💡 Tips

  • 아이콘 색상을 변경하려면 아이콘 페이지에서 편집 버튼을 클릭해 원하는 색으로 바로 수정 후 다운로드할 수 있습니다.
  • 무료 사용 시 출처(Flaticon.com)를 표시해야 합니다. 출처 표시 없이 사용하려면 프리미엄 구독이 필요합니다.
  • 아이콘 팩(Pack) 단위로 다운로드하면 같은 스타일의 아이콘을 일관되게 사용할 수 있어 디자인 통일성이 높아집니다.

자주 묻는 질문

무료로 사용할 수 있나요?

무료로 다운로드 가능하지만 출처(Flaticon.com)를 표시해야 합니다. 출처 표시 없이 상업적으로 사용하려면 프리미엄 구독($9.99/월~)이 필요합니다.

상업적으로 사용할 수 있나요?

무료 플랜은 출처 표시 조건으로 상업 이용 가능합니다. 프리미엄 구독 시 출처 표시 없이 상업 이용이 가능합니다.

SVG 파일도 제공하나요?

네, SVG·PNG·PSD·EPS 형식으로 다운로드할 수 있습니다. SVG는 크기에 상관없이 선명하게 사용할 수 있어 가장 많이 사용됩니다.
