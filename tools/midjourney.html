서비스 소개

Midjourney는 텍스트 프롬프트를 입력하면 그에 맞는 이미지를 생성해 주는 AI 이미지 생성 서비스입니다. 예술적이고 고품질의 이미지 생성에 특화되어 있으며, 디자이너·아티스트·마케터에게 인기가 높습니다. Discord를 통해 사용하거나 웹 인터페이스에서 직접 이용할 수 있습니다.

주요 기능

  • 🖼️ AI 이미지 생성
  • ✏️ 스타일 지정 가능
  • 🔄 이미지 변형·업스케일
  • 📐 종횡비 설정
  • 🎭 다양한 아트 스타일
  • 👁️ 이미지 설명→이미지 변환
  • 💾 고해상도 다운로드
  • 🌐 웹 인터페이스 제공

💡 Tips

  • 프롬프트 끝에 '--ar 16:9' 같은 파라미터로 종횡비, '--v 6' 으로 모델 버전을 지정할 수 있습니다.
  • 원하는 화풍을 구체적으로 쓸수록 좋습니다. 예: 'oil painting, impressionist style, soft lighting'처럼 스타일 키워드를 추가하세요.
  • 생성된 이미지에서 V(Variation) 버튼으로 비슷한 변형을 만들거나, U(Upscale) 버튼으로 고해상도로 확대할 수 있습니다.

자주 묻는 질문

무료로 사용할 수 있나요?

현재 무료 체험 플랜은 제공하지 않습니다. 기본 플랜은 월 $10부터 시작하며, 사용량에 따라 Standard·Pro 플랜을 선택할 수 있습니다.

한국어 프롬프트도 되나요?

한국어 입력도 가능하지만, 영어 프롬프트가 훨씬 더 정확하고 다양한 결과를 만들어냅니다. 영어로 상세하게 설명하는 것을 권장합니다.

생성한 이미지 저작권은 누구에게 있나요?

유료 구독자는 생성한 이미지를 상업적으로 사용할 수 있습니다. 단, Midjourney 이용약관을 반드시 확인하세요.
