서비스 소개

Vercel은 프론트엔드 프로젝트를 위한 클라우드 플랫폼입니다. GitHub·GitLab 저장소와 연동하면 코드 push 시 자동으로 빌드·배포가 이루어집니다. Next.js 개발사이기도 하며 Next.js 프로젝트를 가장 최적화된 환경에서 실행할 수 있습니다. PR마다 미리보기 URL이 생성되어 코드 리뷰 시 결과물을 즉시 확인할 수 있습니다.

주요 기능

  • 🚀 자동 CI/CD 배포
  • 🔗 GitHub·GitLab 연동
  • 👁️ PR 미리보기 URL
  • ⚡ Edge Network CDN
  • 🌐 커스텀 도메인
  • 🔧 환경변수 관리
  • 📊 Analytics
  • 🔒 HTTPS 자동 적용

💡 Tips

  • GitHub 저장소를 Import하면 그 자리에서 1분 내로 첫 배포가 완료됩니다. 설정이 거의 필요 없습니다.
  • PR을 열 때마다 자동으로 미리보기 URL이 생성됩니다. 팀원이나 클라이언트와 코드 없이 결과물을 공유할 수 있습니다.
  • 무료 플랜으로도 커스텀 도메인을 연결하고 HTTPS를 자동 적용할 수 있습니다.

자주 묻는 질문

무료로 사용할 수 있나요?

개인 프로젝트는 Hobby 플랜으로 무료 이용 가능합니다. 팀 협업·고급 분석·보안 기능은 Pro($20/월/사용자~) 플랜이 필요합니다.

Next.js만 지원하나요?

아닙니다. React·Vue·Svelte·Astro·Nuxt 등 주요 프레임워크와 정적 사이트 생성기를 모두 지원합니다. Next.js 최적화가 특히 뛰어날 뿐입니다.

서버리스 함수도 사용할 수 있나요?

네, Vercel Functions(서버리스·엣지 함수)를 통해 API 엔드포인트나 백엔드 로직을 프론트엔드 프로젝트에 포함시킬 수 있습니다.
