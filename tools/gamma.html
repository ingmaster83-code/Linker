서비스 소개

Gamma는 AI를 활용해 프레젠테이션·문서·웹페이지를 자동으로 생성해 주는 서비스입니다. 주제나 개요를 입력하면 AI가 콘텐츠와 디자인을 함께 완성해 줍니다. PowerPoint나 디자인 툴 없이도 빠르게 전문적인 슬라이드 덱을 만들 수 있어 기획·발표·보고서 작성에 유용합니다.

주요 기능

  • 📊 AI 슬라이드 생성
  • 📄 문서·웹페이지 생성
  • 🎨 자동 디자인
  • 🔄 템플릿 제공
  • 🖼️ 이미지 자동 삽입
  • 📤 PDF·PPT 내보내기
  • 🔗 링크 공유
  • ✏️ 편집 가능

💡 Tips

  • 주제 한 줄만 입력해도 AI가 전체 슬라이드 덱을 자동 생성합니다. 이후 원하는 부분만 수정하면 됩니다.
  • 기존 문서나 노트를 붙여넣으면 AI가 자동으로 슬라이드 형식으로 변환해 줍니다.
  • 생성 후 각 슬라이드를 클릭해 텍스트·이미지·레이아웃을 자유롭게 수정할 수 있습니다.

자주 묻는 질문

무료로 사용할 수 있나요?

무료 플랜에서 AI 크레딧 400개를 제공합니다. 더 많은 크레딧과 고급 기능은 Plus($8/월)나 Pro($15/월)로 이용 가능합니다.

PowerPoint 파일로 내보낼 수 있나요?

유료 플랜에서 PPT·PDF 형식으로 내보내기가 가능합니다. 무료 플랜은 PDF 내보내기와 링크 공유만 지원합니다.

한국어 콘텐츠도 생성 가능한가요?

한국어로 주제를 입력하면 한국어 콘텐츠를 생성합니다. 다만 디자인 품질은 영어 콘텐츠와 동일하게 높습니다.
