서비스 소개

Regex101은 정규식(Regular Expression)을 실시간으로 작성하고 테스트하는 온라인 도구입니다. 패턴을 입력하면 테스트 문자열에서 매칭 결과가 즉시 하이라이트되고, 각 토큰에 대한 상세 설명이 제공됩니다. Python·JavaScript·PHP·Go 등 주요 언어의 정규식 엔진을 선택해 실제 코드 환경과 동일하게 테스트할 수 있습니다.

주요 기능

  • 🔍 실시간 정규식 테스트
  • 📖 토큰 상세 설명
  • ⚡ 다국어 엔진 지원
  • 🔄 치환(Replace) 테스트
  • 📊 매칭 결과 시각화
  • 💾 패턴 저장·공유
  • 📋 코드 생성기
  • 🛠️ 디버거 모드

💡 Tips

  • 정규식 엔진(Python/PCRE/JavaScript 등)을 실제 사용할 언어에 맞게 선택하세요. 엔진마다 지원 문법이 다릅니다.
  • 우측 'Explanation' 패널에서 각 정규식 토큰의 의미를 바로 확인할 수 있어 학습에도 유용합니다.
  • 'Code Generator' 탭을 활용하면 작성한 정규식을 Python·JS·PHP 등 코드로 즉시 변환해 줍니다.

자주 묻는 질문

완전 무료인가요?

기본 기능은 완전 무료입니다. 패턴 저장 개수 확대 등 일부 고급 기능에 소액 기부 형태의 유료 옵션이 있습니다.

어떤 언어의 정규식을 지원하나요?

PCRE2(PHP), ECMAScript(JavaScript), Python, Golang, Java, .NET 등 주요 언어의 정규식 엔진을 지원합니다.

작성한 정규식을 저장할 수 있나요?

계정 없이도 공유 링크를 생성할 수 있습니다. 계정을 만들면 작성한 패턴을 라이브러리에 저장할 수 있습니다.
