서비스 소개

Figma는 브라우저에서 동작하는 UI/UX 디자인 툴입니다. 별도 설치 없이 팀원과 실시간으로 동일한 파일에서 협업할 수 있으며, 프로토타입 제작·디자인 시스템·개발자 핸드오프까지 한 툴에서 처리할 수 있습니다. 현재 업계 표준 디자인 툴로 자리잡았습니다.

주요 기능

  • 🖌️ 벡터 디자인
  • 👥 실시간 협업
  • 📱 프로토타입 제작
  • 🔧 컴포넌트·변수
  • 📐 Auto Layout
  • 🔗 개발자 핸드오프
  • 🔌 플러그인 생태계
  • 📁 팀 라이브러리

함께 보면 좋은 사이트

🖼️Canva 📷Unsplash ⭐Flaticon 🔡Google Fonts

💡 Tips

  • Auto Layout을 활용하면 요소 크기가 바뀌어도 레이아웃이 자동으로 조정됩니다. 반응형 컴포넌트 제작에 필수입니다.
  • Figma Community에서 무료 UI 키트·아이콘·템플릿을 찾아 프로젝트에 바로 복사할 수 있습니다.
  • 개발자 모드(Dev Mode)에서 디자인 요소의 CSS 속성, 크기, 색상 코드를 바로 복사할 수 있습니다.

자주 묻는 질문

무료로 사용할 수 있나요?

무료 플랜에서 최대 3개 파일·2명 편집자까지 사용 가능합니다. 팀 협업 확대는 Professional($15/편집자/월~)이 필요합니다.

Sketch나 Adobe XD와 다른 점은?

Figma는 브라우저 기반이라 Mac·Windows 모두 동일하게 사용하고, 실시간 협업이 기본 지원됩니다. Sketch는 Mac 전용, Adobe XD는 서비스가 종료됐습니다.

오프라인에서도 사용할 수 있나요?

Figma 데스크톱 앱을 설치하면 오프라인에서도 편집 가능합니다. 단, 실시간 협업은 인터넷 연결이 필요합니다.
