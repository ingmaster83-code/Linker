서비스 소개

MDN Web Docs(MDN)는 Mozilla가 운영하는 웹 표준 기술 문서 플랫폼입니다. HTML·CSS·JavaScript·Web API 등 모든 웹 기술에 대한 정확하고 상세한 레퍼런스를 제공합니다. 단순 API 문서를 넘어 실습 예제·브라우저 호환성 표·튜토리얼까지 갖추고 있어 입문자부터 시니어 개발자까지 모두 참고하는 사이트입니다.

주요 기능

  • 📄 HTML 레퍼런스
  • 🎨 CSS 레퍼런스
  • ⚡ JavaScript 레퍼런스
  • 🌐 Web API 문서
  • 🔢 브라우저 호환성 표
  • 💻 인터랙티브 예제
  • 📚 학습 가이드
  • 🌍 다국어 번역

💡 Tips

  • 특정 CSS 속성이나 JS 메서드를 구글에서 검색할 때 'mdn flex' 처럼 앞에 'mdn'을 붙이면 MDN 문서로 바로 이동합니다.
  • 각 API 문서 하단의 '브라우저 호환성' 표로 지원 브라우저와 버전을 바로 확인할 수 있습니다.
  • 'Try it' 영역에서 코드를 직접 수정하며 실시간으로 결과를 확인할 수 있습니다.

자주 묻는 질문

무료로 사용할 수 있나요?

완전 무료입니다. Mozilla 재단이 운영하는 비영리 서비스로 광고 없이 이용할 수 있습니다.

한국어 문서도 있나요?

주요 문서는 한국어로 번역되어 있습니다. URL에서 /ko/를 포함하면 한국어 버전으로 볼 수 있으며, 번역되지 않은 일부 문서는 영어로 표시됩니다.

W3Schools와 어떻게 다른가요?

MDN은 Mozilla가 공식으로 관리하는 표준 레퍼런스로 정확도가 높고, W3Schools는 입문자 친화적 튜토리얼 중심입니다. 정확한 스펙 확인은 MDN이 더 신뢰할 수 있습니다.
