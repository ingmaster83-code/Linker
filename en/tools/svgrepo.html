About

SVG Repo hosts over 500,000 SVG icons and illustrations. Most are available under CC0 (public domain) or permissive licenses suitable for commercial use. An in-browser editor lets you change colors before downloading, and you can copy the raw SVG code to embed directly in HTML.

Key Features

  • 🔷 500K+ SVG Icons
  • 🆓 Mostly CC0 License
  • ✏️ Online Color Editing
  • 📋 Copy SVG Code
  • 🔍 Category & Style Search
  • 📦 Batch Downloads
  • 🎨 Color Palette Apply
  • 🌐 Built-in SVG Editor

💡 Tips

  • Click 'Edit SVG' on any icon page to change colors in-browser, then download the modified file or copy the code.
  • Use the 'SVG Code' button to copy the markup and paste it inline into HTML for icon fonts that inherit CSS color.
  • Always check the license badge on each icon — most are CC0 but a few require attribution.

Frequently Asked Questions

Is it free?

Most icons are CC0 (public domain) and completely free. Some icons carry other licenses — always check the license badge before downloading.

Can I use them commercially?

CC0 icons can be used for any commercial purpose. The license type is displayed on every icon page.

Is PNG available too?

PNG conversion is offered, but SVG is strongly recommended for quality and scalability at any size.
