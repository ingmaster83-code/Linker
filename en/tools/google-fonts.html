About

Google Fonts is Google's free web font service offering over 1,500 font families. Embed any font with a single @import or link tag, and load only the specific weights you need to keep page loads fast. All fonts are under open licenses (OFL and others), free for personal and commercial use across web, print, and video.

💡 Tips

  • When selecting a Korean font, make sure to check the 'Korean' subset — the Latin-only subset won't render Korean characters correctly.
  • For web performance, only select the font weights you actually use. Each extra weight adds to page load time.
  • Use the 'Pairings' tab on any font page to get Google's recommended font combinations for headings and body text.

Frequently Asked Questions

Is it completely free?

Yes, entirely free for personal and commercial projects across all media — web, print, video, and more.

Can I download font files?

Yes — TTF and OTF files are available for download from each font page for local installation or offline projects.

Does it include non-Latin scripts?

Yes, it includes Korean (Nanum Gothic, Noto Sans KR, etc.), Japanese, Arabic, Cyrillic, and many other scripts.
