About

Figma is a browser-based UI/UX design tool that runs on any OS without installation. Teams can collaborate on the same file in real time, build design systems with shared components and variables, create interactive prototypes, and hand off specs to developers — all in one place.

Key Features

  • 🖌️ Vector Design
  • 👥 Real-time Collaboration
  • 📱 Prototyping
  • 🔧 Components & Variables
  • 📐 Auto Layout
  • 🔗 Dev Handoff
  • 🔌 Plugin Ecosystem
  • 📁 Team Libraries

💡 Tips

  • Use Auto Layout so components resize and reflow automatically — essential for building responsive, reusable UI elements.
  • Browse Figma Community for free UI kits, icon sets, and templates you can duplicate directly into your project.
  • In Dev Mode, developers can copy CSS properties, dimensions, and color codes from any design element with one click.

Frequently Asked Questions

Is Figma free?

The free plan supports up to 3 files and 2 editors. Larger team collaboration requires the Professional plan starting at $15/editor/month.

How does it compare to Sketch or Adobe XD?

Figma is browser-based, works on Mac and Windows equally, and has built-in real-time collaboration. Sketch is Mac-only, and Adobe XD has been discontinued.

Does it work offline?

Yes — install the Figma desktop app for offline editing. Real-time collaboration requires an internet connection.
