About

Can I Use is a browser compatibility database that tells you exactly which versions of Chrome, Firefox, Safari, Edge, iOS Safari, and other browsers support a given web technology. It shows global usage share percentages alongside compatibility data, helping you make informed decisions about which features to use or polyfill.

Key Features

  • 📊 Browser Compatibility Tables
  • 🔍 Feature Search
  • 📱 Mobile Browsers Included
  • 📈 Global Usage Share
  • ⚠️ Partial Support Notes
  • 🔗 MDN Integration
  • 📅 Version History
  • 🆓 Completely Free

💡 Tips

  • For any CSS or JS feature, check the 'Global usage' percentage to understand what share of real-world users are affected by a missing feature.
  • Yellow cells mean partial support or vendor prefix required — click the cell for specific caveats and workarounds.
  • In Settings, exclude browsers you don't support to get an accurate usage share figure for your actual target audience.

Frequently Asked Questions

Is it free?

Completely free. It's an open data project funded by ads and community contributions.

How often is the data updated?

Data is updated continuously as new browser versions are released, typically staying current within a few weeks of a major release.

Does it include mobile browsers?

Yes — Chrome for Android, iOS Safari, Samsung Internet, and other major mobile browsers are all included.
